GLOBAL AZURE BOOTCAMP 2016

O Global Azure BootCamp, é um evento gratuito realizado anualmente em escala global, para que o conhecimento sobre Microsoft Azure seja disseminado entre a comunidade, incentivando a sua utilização e adoção.

Público Alvo

Desenvolvedores, arquitetos e it-pros com interesse em conhecer e aprender mais sobre a plataforma de Cloud Computing da Microsoft.

Networking

Aproveite para conhecer pessoas, fazer crescer sua rede de contatos que pode lhe proporcionar valor, desde a troca de valiosas informações sobre soluções na Nuvem, ou simplesmente para um bom papo sobre tecnologia.

Plug-se a Nuvem

Um dia de imersão completa que vai te deixar apto (pro) a desenvolver aplicações e/ou criar sua infraestrutura utilizando recursos de Cloud Computing no Microsoft Azure.

Abertura do Evento

Armazenando Imagens da Nuvem

09:20 - 10:00

Armazenando Imagens da Nuvem

Como funciona o armazenamento de arquivos na nuvem? Vamos botar isso na prática armazenando fotos na nuvem!? Explorando um pouco do potencial de armazenar blobs no Windows Azure, e entender os conceitos de contêineres de armazenamento e hierarquias.

Comece a utilizar o Azure Scheduler

DevOps Deep Dive / DevOps in action

10:40 - 11:20

DevOps Deep Dive / DevOps in action

A sessão terá início com uma introdução do Release Management, seguido de uma demonstração de ponta a ponta sobre Deploy e Testes de uma aplicação ASP.NET MVC 6 em um container Docker em uma máquina Linux no Azure usando Release Management no VSTS. Utilizaremos a mais recente versão do Release Management no Visual Studio Team Services.
