O Global Azure BootCamp, é um evento gratuito realizado anualmente em escala global, para que o conhecimento sobre Microsoft Azure seja disseminado entre a comunidade, incentivando a sua utilização e adoção.

Público Alvo

Desenvolvedores, arquitetos e it-pros com interesse em conhecer e aprender mais sobre a plataforma de Cloud Computing da Microsoft.

Networking

Aproveite para conhecer pessoas, fazer crescer sua rede de contatos que pode lhe proporcionar valor, desde a troca de valiosas informações sobre soluções na Nuvem, ou simplesmente para um bom papo sobre tecnologia.

Plug-se a Nuvem

Um dia de imersão completa que vai te deixar apto (pro) a desenvolver aplicações e/ou criar sua infraestrutura utilizando recursos de Cloud Computing no Microsoft Azure.

    Como publicar uma aplicação AspNetCore em um AppService do Azure com Linux rodando Docker

    Como monitorar minha aplicação
    Uma das principais práticas de DevOps é monitorar sua aplicação em produção para saber como ela se comporta. Nesta palestra vamos usar o serviço do Azure chamado Application Insights para coletar a telemetria de uma aplicação e como os desenvolvedores podem usar essas informações para solucionar problemas e melhorar seu código.
    Leandro Prado

    Azure e Raspberry Pi: um exemplo de monitoramento remoto
    Integração de dados com a nuvem é uma exigência cada vez mais comum em projetos embarcados. O objetivo da palestra é demonstrar como é possível monitorar através do Azure dados gerados pelo Raspberry PI.
    Ângelo Polotto
